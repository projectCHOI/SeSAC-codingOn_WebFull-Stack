*Bootstrap
- 반응형, 모바일 우선 웹 사이트를 빠르게 디자인하고 커스터마이징할 수 있게 도와주는 오픈 소스 프론트엔드 프레임워크
- 네비게이션 바, 모달, 캐러셀 등의 다양한 CSS 및 JavaScript 컴포넌트가 제공
- 그리드 시스템을 통해 레이아웃 설계가 가능

*Bootstrap의 주요 특징
- 그리드 시스템: Bootstrap은 12열 반응형 그리드 시스템을 사용하여 화면 크기에 따라 자동으로 레이아웃이 조정
- 반응형 디자인: 미디어 쿼리를 통해 화면 크기에 맞춰 레이아웃을 자동으로 조정
- 미리 스타일된 컴포넌트: 버튼, 폼, 카드, 네비게이션 바 등 다양한 컴포넌트가 일관된 스타일로 제공
- JavaScript 플러그인: 모달, 드롭다운, 툴팁 등의 인터랙티브한 요소를 위한 JS 플러그인이 포함
- 커스터마이징 가능: 변수를 수정하거나 직접 CSS를 추가하여 Bootstrap 스타일을 쉽게 변경 가능
