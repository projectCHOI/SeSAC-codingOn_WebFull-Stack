문자열 관련 메서드

속성 및 메서드 활용

  • length, toUpperCase, toLowerCase, trim 등의 문자열 속성과 메서드를 통해 문자열의 길이 확인, 대소문자 변환, 공백 제거 등을 실습

    • 문자열 메서드

  • indexOf, charAt, slice 등을 사용해 특정 문자나 문자열의 위치를 찾거나, 문자열의 특정 부분을 추출하는 방법

    • 문자열 변환 및 대체

  • replace, replaceAll, split 등을 통해 문자열에서 특정 문자를 교체하거나, 특정 문자를 기준으로 분리해 배열로 변환

    • 배열 관련 메서드

    배열 요소 추가 및 제거

  • push, pop, unshift, shift를 사용해 배열에 요소를 추가하거나 제거하는 방법.

    • 기타 배열 메서드

  • includes로 특정 요소가 배열에 포함되어 있는지 확인하고, indexOf로 요소의 인덱스를 찾는 방법
  • reverse로 배열 순서를 뒤집고, join을 사용해 배열을 문자열로 병합하는 방법

    • 반복문을 활용한 배열 순회

    for 반복문

  • 기본 for 문과 for of 문을 사용해 배열의 요소들을 순회

    • forEach 메서드

  • forEach를 사용해 배열의 요소와 인덱스에 접근하며 반복문을 통해 요소를 출력

    • 고급 배열 메서드

    filter, find, map 메서드

  • filter를 통해 조건을 만족하는 요소만을 새로운 배열로 반환하고, find는 조건을 만족하는 첫 번째 요소를 찾기
  • map을 사용해 배열의 각 요소에 대해 특정 작업을 수행하여 새로운 배열

    • 객체 순회

    객체의 키와 값 순회

  • for...in 문을 사용해 객체의 키에 접근하고, 대괄호 표기법을 통해 각 키의 값을 출력
