반복문 for문

반복문 예제

  • i++처럼 1씩 증가, i += 2로 2씩 증가, 그리고 i--로 감소

    • 배열 순회

  • for 반복문을 이용해 fruits나 arr 배열을 순회하는 방식
  • fruits.length 속성을 사용해 배열의 끝에서 반복이 멈추도록 설정

    • 숫자 합 구하기

  • 이런저런 방식의 합

    • 중첩 반복문

  • 구구단을 위해 중첩 반복문을 잘 사용
  • 중첩 반복문은 그리드나 표, 다차원 데이터 구조를 다룰 때 유용

    • Continue 문

  • continue 문을 사용해 짝수일 때 건너뛰고 홀수만 합산하는 방식
  • 반복문에서 특정 조건을 건너뛰고 싶을 때 유용

    • while 문 사용

  • n1과 n2 변수를 사용해 while 문을 활용하여 특정 조건에서만 반복 실행

    • 무한 루프와 break 문

  • while (true)로 무한 루프를 생성하고, 특정 조건(a > 10)에서 break를 사용해 루프를 종료

    • 사용자 확인 대화상자

  • confirm 함수로 반복 여부를 사용자에게 묻기
  • 사용자가 "확인"을 선택하면 true, "취소"를 선택하면 false가 반환되어 반복을 중단
  • 각 반복마다 alert 창으로 몇 번째 반복인지 확인 가능
